# modular-redux-design-pattern

Encapsulate redux states and actions.

## Structure

```bash
src
├── App.js
├── App.test.js
├── components
│   └── TodoList
    │   ├── index.jsx
    │   └── store.js # Sub store
├── index.js
├── reportWebVitals.js
├── setupTests.js
└── store
    └── index.js # Project store
```

## How to run this project locally

1. Clone this repo
2. Run `npm install`
3. Run `npm start`
